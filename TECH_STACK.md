# 技术栈

## 运行环境

- 浏览器运行时。
- 可选使用本地静态服务器做验证。

## 框架和库

- HTML、CSS、Vanilla JavaScript。
- MVP 不使用包管理器依赖。

## 命令

```powershell
# 安装：
# 无需安装。

# 运行：
python -m http.server 4173

# 测试：
# 打开 http://localhost:4173 并运行示例分析流程。
```

## 环境变量

- MVP 无环境变量。
- 后续可选 API 版本只能在服务端使用 `OPENAI_API_KEY`。

## 外部服务

- MVP 不依赖外部服务。
- 目标部署平台是 GitHub Pages。
